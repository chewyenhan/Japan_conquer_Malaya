## 📖 游戏简介

**「马来亚1941：抉择的十字路口」** 是一款专为**马来西亚华文独中高三历史**课堂设计的沉浸式角色扮演游戏。

游戏基于课本《马来西亚史》中「日本侵略马来半岛」章节。玩家扮演一位1941年的年轻民族主义领袖（KMM成员），在日军入侵、英军溃退、政治真空的历史十字路口做出抉择。每个选择都将影响三项评分——**独立契机、民族代价、占领者信任**——并最终导向四种历史结局之一。

## 🎮 怎么玩

### 快速开始

1. 打开 [游戏页面](https://chewyenhan.github.io/Japan_conquer_Malaya/)
2. 选择分组数量（支持 **1-6 组**课堂竞赛）
3. 为每组取名（或使用默认组名）
4. 依次完成 **4 个历史关卡**，每关做出 1 次抉择
5. AI 即时生成 **「统考视角 · 历史纵深剖析」**
6. 根据最终评分判定结局，查看 **统考复习卡**

### 游戏流程

```
开始 → 分组设置 → 命名 → 关卡1 → 关卡2 → 关卡3 → 关卡4 → 结局
                                 ↓          ↓          ↓          ↓
                              AI分析     AI分析     AI分析     复习卡
```

| 步骤 | 内容 | 用时 |
|------|------|------|
| 🏁 设置 | 选组数、取名 | 1分钟 |
| ⚡ 关卡1 | 山雨欲来（日军登陆） | 2-3分钟 |
| 🚢 关卡2 | 白人的溃败（英军败退） | 2-3分钟 |
| 🌴 关卡3 | 丛林闪电战（日军军力） | 2-3分钟 |
| 🏭 关卡4 | 共荣的真相（资源掠夺） | 2-3分钟 |
| 🏆 结局 | 评分揭晓 + 统考复习卡 | 2-3分钟 |

## 📊 评分系统

### 三维评分

| 维度 | 含义 | 范围 |
|------|------|------|
| 🆓 **独立契机** | 为民族独立积累的能量与时机 | 0~12 |
| 💔 **民族代价** | 民众付出的牺牲与资源的流失 | 0~12 |
| 🎌 **占领者信任** | 日军对你的信任度 | -7~12 |

### 四种结局

| 结局 | 触发条件 | 评级 |
|------|----------|------|
| 🏆 **民族觉醒者** | 独立契机 ≥ 7 且 信任 ≤ 3 | ⭐ 优秀 |
| 🎭 **两难幸存者** | 默认（中庸之道） | ⭐ 良好 |
| ⛓️ **傀儡统治者** | 信任 ≥ 6 且 独立契机 ≤ 4 | ⚠️ 反思 |
| 💀 **理想的殉道者** | 民族代价 ≥ 7 | 💪 勇气可嘉 |

## 📚 课文考点覆盖

每个关卡精确对应统考课文考点：

| 关卡 | 对应考点 |
|------|----------|
| ⚡ 第一关 | **沦陷原因(二)**：日本利用民族矛盾，以"解放者"旗号入侵 |
| 🚢 第二关 | **沦陷原因(一)**：英军准备不足，殖民体制排斥本地人 |
| 🌴 第三关 | **沦陷原因(三)**：日军丛林训练、中国战场经验、先发制人 |
| 🏭 第四关 | **太平洋战争原因(I)**：资源控制 & 大东亚共荣圈的真相 |

涵盖课文全部 7 大考点：
- 太平洋战争原因：资源控制 & 大东亚共荣圈
- 登陆时间地点：哥打峇鲁、宋卡、北大年
- 关键事件：威尔斯太子号 & 击退号被击沉
- 沦陷原因(一)(二)(三)
- 北婆罗洲 & 砂拉越被占

## 🛠️ 技术架构

```
index.html (单文件 ~1200行)
├── HTML/CSS——暗色调军事主题 UI
├── Vanilla JS——无构建工具，零依赖
├── Gemini API——AI 生成统考视角分析
├── Cloudflare Worker 代理——安全中转 API Key
├── localStorage——保存进度（支持断点续玩）
├── SpeechSynthesis——浏览器内置 TTS 语音朗读
└── 多组支持——1-6 组课堂竞赛模式
```

### API 模式

| 模式 | 说明 |
|------|------|
| 🌐 **Worker 代理（默认）** | 通过 `malaya1941-ai.chewyenhan.workers.dev` 中转，学生无需 Key |
| 🔑 **直连模式** | 教师可在设置中填入自己的 Gemini API Key |
| 📖 **离线回退** | API 不可用时自动使用预写分析，游戏完全可用 |

## 🏫 课堂使用建议

1. **投影模式**：教师电脑打开游戏，投屏到教室大屏，全班讨论共同抉择
2. **分组模式**：1-6 组各自完成 → 对比结局 → 讨论「为什么不同选择导致不同命运」
3. **个人模式**：每位学生用手机打开链接独自完成 → 分享结局 → 课堂讨论
4. **配合统考复习**：游戏结束后利用「统考复习卡」逐条讲解 7 大考点
