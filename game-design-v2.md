# 马来亚1941：抉择的十字路口 — 课堂版游戏设计

> 面向：高三学生（18岁）| 时长：~15分钟 | 模式：5关选择题 + AI历史分析

---

## 一、设计原则

| 约束 | 应对 |
|------|------|
| **18岁高三学生** | 选项不设明显的"对/错"，每个选择都有代价。考验学生的历史思辨能力，而非记忆背诵。 |
| **课堂时间有限** | 总共5关，每关约3分钟（读题30秒+选择30秒+看AI分析2分钟），一节课可完成。 |
| **简单关卡制** | 5个关卡 = 5个历史关键节点。每关3选1，选后AI即时生成分析。无需复杂资源管理。 |

---

## 二、5关结构（贯穿课文全部考点）

```
关卡1 ──→ 关卡2 ──→ 关卡3 ──→ 关卡4 ──→ 关卡5
 山雨欲来    白人的溃败   丛林闪电战   共荣的真相   历史的回声
 (Dec 8)    (Dec 10)    (Late Dec)   (Jan 1942)   (尾声)
```

### 关卡1：山雨欲来 ⚡
**时间**：1941年12月8日凌晨

**场景**：日军在哥打峇鲁登陆。一个日本军官带着《大东亚共荣宣言》和经费找到你，请你号召民众"欢迎解放者"。同时，英国殖民官员要求你组织青年协助盟军。

**课文考点**：沦陷原因(二) — 日本以"反对白人殖民主义"旗号入侵，利用民族矛盾

| 选项 | 后果 |
|------|------|
| A. 接受日军援助，号召人民"迎接亚洲解放" | 独立契机+2 / 占领者信任+2 / 民族代价+1 |
| B. 表面中立，私下同时接触双方 | 独立契机+1 / 占领者信任+1 / 民族代价+0 |
| C. 协助英军，相信战后英国会给予自治 | 独立契机+0 / 占领者信任-1 / 民族代价+1 |

---

### 关卡2：白人的溃败 🚢
**时间**：1941年12月10日

**场景**：「威尔斯太子号」和「击退号」在关丹海域被日军飞机击沉。英军皇家空军基地被炸毁。日军完全掌握制海权和制空权。英军官员慌乱地请求你帮忙疏散，却连基本的地图都没有。

**课文考点**：沦陷原因(一) — 英军不认真准备，殖民体制排斥本地人参与防务

| 选项 | 后果 |
|------|------|
| A. 帮助英军撤退，换取未来政治承诺 | 独立契机+1 / 占领者信任-2 / 民族代价+0 |
| B. 拒绝英军，冷眼旁观殖民者的溃败 | 独立契机+2 / 占领者信任+1 / 民族代价+0 |
| C. 趁机组织本地自卫队，不靠任何人 | 独立契机+3 / 占领者信任-1 / 民族代价+2 |

---

### 关卡3：丛林闪电战 🌴
**时间**：1941年12月下旬

**场景**：日军以惊人速度穿越丛林向南推进。你亲眼看到：日军士兵在夜间行军如履平地，利用自行车在橡胶林间快速机动。这与英军的笨拙形成鲜明对比。一名日军上尉得意地告诉你，这些部队在中国战场历练多年。

**课文考点**：沦陷原因(三) — 日军丛林训练+夜间作战+中国战场经验+先发制人+周密计划

| 选项 | 后果 |
|------|------|
| A. 钦佩日军的军事效率，决定全力配合 | 独立契机+0 / 占领者信任+3 / 民族代价+2 |
| B. 警惕：如此高效的军队，战后不会轻易离开 | 独立契机+2 / 占领者信任+0 / 民族代价+0 |
| C. 尝试学习日军战术，暗中为将来做准备 | 独立契机+3 / 占领者信任-1 / 民族代价+1 |

---

### 关卡4：共荣的真相 🏭
**时间**：1942年1月，马来亚即将全面沦陷

**场景**：日军已控制大部分城镇。你注意到：橡胶和锡矿正被系统性地运往日本；"共荣圈"的宣传背后，本地物资被无偿征用，米粮短缺开始出现。当初的"解放"承诺正在变质——日本来此的真正目的是**资源**。

**课文考点**：太平洋战争原因(I) — 日本资源短缺，发动战争为控制东南亚资源，建立"自给自足"的大东亚共荣圈

| 选项 | 后果 |
|------|------|
| A. 继续合作——现实政治，生存优先 | 独立契机+0 / 占领者信任+3 / 民族代价+3 |
| B. 暗中记录日军掠夺行径，保存证据 | 独立契机+2 / 占领者信任+0 / 民族代价+1 |
| C. 公开质疑日军的"共荣"承诺 | 独立契机+1 / 占领者信任-3 / 民族代价+3 |

---

### 关卡5：历史的回声 📜
**时间**：1942年1月31日，马来半岛全面沦陷。以及战后。

**场景**：游戏终局。AI 根据你的 5 关选择，生成：
1. **结局判定**（4种之一）
2. **统考复习卡**（汇总你在游戏中遇到的所有课文考点）
3. **历史真相**：现实中 KMM 等民族主义组织的结局——他们确实接受了日本的合作，但战后被视为通敌者，独立梦想并未通过这条路实现。

**课文考点**：全部 — 统考视角回顾

**4种结局**（基于累计评分）：
| 结局 | 条件 | 描述 |
|------|------|------|
| 🏆 民族觉醒者 | 独立契机≥8, 占领者信任≤5 | 你保持了清醒。战后你成为独立运动的中坚力量，用亲身经历揭露殖民与侵略的双重面目。 |
| 🎭 两难幸存者 | 三项均衡 | 你在夹缝中保全了组织和民众。但历史证明：没有真正独立之前，所有"生存策略"都只是延迟了选择的到来。 |
| ⛓️ 傀儡统治者 | 占领者信任≥9, 独立契机≤3 | 你成为日本军政府的合作者。战争结束后，你被英国殖民当局审判为"通敌者"。历史对你做出了严厉的评判。 |
| 💀 理想的殉道者 | 民族代价≥10 | 你的理想主义让你过早暴露。无论你的初衷多么高尚，在两大帝国机器的碾压下，个人的力量微乎其微。 |

## 五、技术方案

基于仓库已有模式（美国独立战争 / 马六甲）：

- **文件**：单个 `index.html` + `style.css`（可选）
- **前端**：原生 HTML/CSS/JS（不依赖框架，降低复杂度）
- **AI**：Gemini API（通过内置代理，学生无需配置 API key）
- **多人**：支持 1-6 组同时游戏（课堂分组竞赛，看哪组的结局最好）
- **语音**：可选 TTS 朗读关卡文本
- **存储**：localStorage 保存进度
